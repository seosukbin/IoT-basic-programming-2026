 ### 프로그램 개요
 - 프로그램이란 사용자의 요구에 따라 데이터를 처리(표현, 이동,저장,로드)할수 있는 소프트 웨어
 - 프로그램을 만드는 작업: 프로그래밍(코딩,SW개발)

 ###c언어
 - 켄 톰슨과 데니스 리치가 unix 운영 체제를 개발하기 위해서 만든 언어
 - c++,Java, Python등 모든 언어의 기본이 되는 언어
 - 1999년에 표준에 따라서 학습

 ### 컴파일과 컴파일  사용법

- c, cpp -> 확장자 exe 파일로 생성
